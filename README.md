# 📊 Student Performance Analysis

## 📌 Project Overview

This project analyzes student performance using statistical techniques to understand how study habits impact academic results. It focuses on data distribution, variability, outlier detection, and hypothesis testing to draw meaningful conclusions.

---

## 🎯 Objectives

* Analyze the distribution of student marks
* Calculate key statistical measures (mean, median, standard deviation)
* Detect outliers using the IQR method
* Examine the relationship between study hours and marks
* Perform hypothesis testing to validate whether study time significantly affects performance

## 📊 Analysis Workflow

1. Data Generation & Loading
2. Descriptive Statistics (Mean, Median, Std Dev)
3. Data Distribution Visualization
4. Outlier Detection using IQR
5. Correlation Analysis
6. Hypothesis Testing (t-test)
7. Insights & Conclusion

---

## 📈 Key Insights

* Student marks show moderate variability across the dataset
* Median is more reliable than mean when outliers are present
* A positive relationship exists between study hours and marks
* Outliers were successfully identified using the IQR method
* Hypothesis testing indicates that study hours have a statistically significant impact on performance
